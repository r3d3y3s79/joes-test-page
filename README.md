## Setup Instructions

1. **Clone the Repository:**
2.    - Open your terminal.
      -    - Run `git clone https://github.com/r3d3y3s79/joes-test-page.git` to clone the repository.
       
           - 2. **Navigate to the Directory:**
             3.    - Use `cd joes-test-page` to enter the project directory.
               
                   - 3. **Open in Browser:**
                     4.    - Open `index.html` in your preferred web browser to view the page.
                       
                           - ## Usage Examples
                       
                           - - **View the Page:** Simply open `index.html` in any web browser after cloning the repository to see the demo page.
                             - - **Modify Content:** Edit `index.html` and `styles.css` to customize the page according to your needs.
                              
                               - ## Contribution Guide
                              
                               - 1. **Fork the Repository:**
                                 2.    - Use the "Fork" button at the top right of the GitHub page to create your own copy of the repository.
                                   
                                       - 2. **Create a New Branch:**
                                         3.    - Navigate to your forked repository in your terminal.
                                               -    - Run `git checkout -b my-new-feature` to create a new branch.
                                                
                                                    - 3. **Make Your Changes:**
                                                      4.    - Edit files and commit your changes using `git commit -m 'Add my new feature'`.
                                                        
                                                            - 4. **Push to GitHub:**
                                                              5.    - Run `git push origin my-new-feature` to push your changes to GitHub.
                                                                
                                                                    - 5. **Submit a Pull Request:**
                                                                      6.    - Go to the original repository and click "New Pull Request" to submit your changes for review.
